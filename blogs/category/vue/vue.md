# Vue3

__使用vite搭建vue项目__

```js
npm init @vitejs/app

yarn create @vitejs/app
```
__使用模板__
```js
npm init @vitejs/app my-vue-app --template vue

// npm 7+, 需要额外的双横线：
npm init @vitejs/app my-vue-app -- --template vue

// yarn
yarn create @vitejs/app my-vue-app --template vue
```
